使用 Hexo 和 Github 搭建 Next 主题个人博客

发表于 2021-08-25 更新于 2021-08-26

本文详细描述了如何在 Github 上，使用 Hexo 部署博客，并配置个性化 Next 主题。

一、安装

1、安装Node.js

  • Node.js官方下载地址
  • Node.js安装教程

2、安装Git

  • Git官方下载地址

  • Git安装教程

  • 注册Github账号

    • 教程
    • 输入用户名，Email，密码。比如，我的用户名是：lyfdnrjbrs

  • 创建博客仓库

    • 教程
    • 注意，仓库名应该为：用户名.github.io。比如，我的仓库名是：lyfdnrjbrs.github.io

  • 配置SSH Key

    • 打开Git Bash终端

    • 设置user.name和user.email

    • 生成SSH Key

      此时，在用户文件夹下就会有一个新的文件夹.ssh，里面有刚刚创建的ssh密钥文件id_rsa和id_rsa.pub。

    • 获取SSH Key

      如下图


    • 在GitHub中添加SSH Key

      • 打开 GitHub，点击头像，选择Settings。
      • 选择SSH and GPG keys，点击New SSH key。
      • 将之前获取到的id_rsa.pub的内容复制到Key文本框中。
